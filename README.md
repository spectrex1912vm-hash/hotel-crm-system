Hotel CRM System — QA Portfolio



Project Description



This project is a QA portfolio for a simple Hotel CRM system.



The system allows managing hotel bookings, including:

\- Creating bookings

\- Editing bookings

\- Deleting bookings

\- Viewing bookings in calendar

\- Basic room and guest data handling



Main focus of this project is \*\*manual QA testing and API testing practice\*\*.



\---



QA Artifacts



This repository contains 3 main QA documents:



\- Test Cases (`TestCases.md`)

\- Bug Reports (`BugReports.md`)

\- Checklists (`Checklists.md`)
